**Query2Insight AI**
**GenAI-Powered Business Analytics Copilot**
**Overview**

Query2Insight AI is a GenAI-powered analytics system that enables users to interact with databases using natural language. It converts user queries into SQL, executes them on a database, and generates actionable business insights using LLMs.

This project demonstrates the integration of AI + Data + Full Stack concepts to build an intelligent analytics assistant for real-world enterprise use cases.

**Features**
 Convert natural language → SQL queries
 AI-powered insight generation from query results
 Interactive data visualization using Streamlit
 Real-time query execution on database
 Modular architecture (LLM + DB + UI layers)
📈Business decision support system
